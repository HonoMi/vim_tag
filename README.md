# vim_tag
Tag the itemization lines of markdown.
Only lines which starts with "-" or "*"(i.e. the itemization of markdown) are the targets.  

## how to use
[主張]
```
    :Message
```

[根拠]
```
    :Argument
```

[例]
```
    :Example
```

[推測]
```
    :Conjecture
```

[事実]
```
    :Fact
```

[引用]
```
    :Quote
```

[原因]
```
    :Cause
```

[!] (means todo!)
```
    :ToDo
```

[done]
```
    :Done
```

Reset all the tags attached to the line.
```
    :Reset
```
